# Blinkit Business Analytics Dashboard

## Project Overview

This initiative evaluates Blinkit's business performance through the use of MySQL, Power BI, Power Query, DAX, and Excel. The dashboard delivers valuable insights into sales, customer behavior, operations, inventory, delivery efficiency, and customer feedback, empowering informed, data-driven decisions.

---

## Tools & Technologies

- MySQL  
- Power BI  
- DAX  
- Power Query  
- Microsoft Excel  

---

## Dataset

The project incorporates several interconnected datasets structured with a Star Schema, including:

- Customers  
- Orders  
- Order Items  
- Products  
- Inventory  
- Delivery Performance  
- Marketing Performance  
- Customer Feedback

## How to Run

1. Import the provided datasets into MySQL.  
2. Run the SQL queries in `SQL_Analysis_Queries.sql`.  
3. Open `blinkit_dashboard.pbix`.  
4. Refresh data connections if necessary.
